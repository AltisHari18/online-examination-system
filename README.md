# Online Examination System

A web-based online examination system developed using:

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Features
- Student & Teacher login
- Online MCQ exams with timer
- Auto submission
- Result calculation
- Teacher result view & Excel export
- PDF marksheet download

## How to Run
1. Install XAMPP
2. Copy project to `htdocs`
3. Import database in phpMyAdmin
4. Open `http://localhost/online-exam`
